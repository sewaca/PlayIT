# PlayIT

Play IT - проект факультета ИСИТ СПбГУТ

## Запуск dev-версии фронтенд-приложения:

```
npm run dev
```

## Запуск прокси:

```
npm run proxy
```

## Запуск тестового бекенда:

```
npm run server
```
